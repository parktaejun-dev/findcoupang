# 광고 감지기 (Ad Detector)

검색 결과에서 제휴/광고 링크가 포함된 페이지를 **클릭 전에 미리 감지**하는 Chrome 확장 프로그램입니다.

## 주요 기능

- 🔍 **검색 결과 자동 분석**: Google, Naver, Daum, Bing 검색 결과 페이지에서 블로그/리뷰 링크를 자동으로 분석
- 🏷️ **배지 표시**: 각 링크 옆에 상태 배지 표시
  - 🔴 **광고**: 제휴 마케팅 링크 감지
  - 🟠 **의심**: 단축 URL 감지 (우회 가능성)
  - 🟢 **✓**: 미감지
- ⚙️ **사용자 설정**: 텍스트/아이콘 스타일, 사이트별 활성화/비활성화

## 감지 대상

### 제휴 링크 (확정)
- `coupa.ng`, `link.coupang.com`
- `partners.coupang.com`

### 단축 URL (의심)
- `bit.ly`, `link24.kr`, `vo.la`, `c11.kr`, `me2.do` 등

## 설치 방법

### Chrome 웹 스토어
*(심사 후 링크 추가 예정)*

### 수동 설치 (개발자 모드)
1. 이 저장소를 다운로드/클론
2. Chrome에서 `chrome://extensions` 접속
3. **개발자 모드** 활성화
4. **압축해제된 확장 프로그램을 로드합니다** 클릭
5. 다운로드한 폴더 선택

## 사용 방법

1. 확장 프로그램 아이콘 클릭
2. **미리보기 모드** 활성화
3. 검색 결과 페이지에서 배지 확인

## 권한 안내

| 권한 | 용도 |
|------|------|
| `storage` | 사용자 설정 저장 |
| `<all_urls>` (선택) | 링크된 페이지 내용 분석 |

**참고**: `<all_urls>` 권한은 미리보기 모드를 활성화할 때만 요청되며, 비활성화 시 사용되지 않습니다.
